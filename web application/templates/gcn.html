Graph Convolutional Network Model

GCN (Graph Convolutional Network) is a type of neural network designed to work with graph data, such as brain connectivity measures. The goal of GCN is to learn node-level or graph-level representations of the input graph, which can then be used for a variety of downstream tasks such as node classification or link prediction.

In the context of brain connectivity, GCN can be used to predict the category of an individual (CN, AD, LMCI, SMC, or EMCI) based on their connectivity measures. This can help with diagnosis and understanding of neurodegenerative diseases such as Alzheimer's disease.
