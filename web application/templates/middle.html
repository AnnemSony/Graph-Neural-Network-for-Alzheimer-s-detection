Project Details

This is a web application for benchmarking Graph Neural Network (GNN) models. The application currently supports two GNN models, Graph Convolutional Network (GCN) and Graph Attention Network (GAT).

We have multiple barin connectivity datafiles for 908 patients,each patient brain is divided into 160 regions were each region is taken as a node and connection between the regions were noted as weights.

The categories of the labels in the brain connectivity dataset, with the number of examples for each label, are:

Label Number of Examples Category
0 229 CN: cognitively normal individuals
1 94 AD: individuals with Alzheimer's disease
2 152 LMCI: individuals with mild cognitive impairment
3 222 SMC: individuals with subjective memory complaints
4 210 EMCI: individuals with early mild cognitive impairment

One of the main benefits of GNNs is their ability to model graph-structured data, which is a natural fit for brain connectivity data since it can be represented as a graph with nodes representing brain regions and edges representing the strength of the connectivity between them.

GNNs can be used to learn representations of brain connectivity graphs that capture important features and patterns in the data, which can be used for downstream tasks such as classification of disease states, prediction of clinical outcomes, and identification of biomarkers. GNNs can also be used for clustering analysis, where they can group similar connectivity patterns together and identify subgroups within the dataset. Overall, GNNs provide a powerful tool for analyzing brain connectivity datasets, enabling researchers to gain new insights into brain function and disease.
